ETIQUETA DE ENVIO:
Cómo se genera la etiqueta de cada transportista:
1. Vas al pedido en tu admin de Shopify (Pedidos → seleccionas el pedido sin preparar).
2. Haces clic en "Crear etiqueta de envío" — Shopify se conecta directamente con los transportistas integrados y te muestra las opciones disponibles según tu ubicación de fulfillment.
3. Seleccionas el transportista y el servicio (por ejemplo, DHL Paket, SEUR, Correos, etc.) — Shopify negocia tarifas con descuento directamente con ellos.
4. Confirmas el paquete y el peso, y haces clic en "Comprar etiqueta".
5. La etiqueta se genera al instante y puedes imprimirla en formato 4×6" (impresora térmica) o 8,5×11" (impresora normal).
Puntos clave a tener en cuenta:
* Los transportistas disponibles dependen de tu país de fulfillment — para España los más habituales son Correos, SEUR, DPD, etc.
* Puedes configurar transportistas preferidos en Configuración → Envío y entrega → Etiquetas de envío → Gestionar transportistas preferidos, para que aparezcan primero por defecto.
* Si usas un transportista que no está integrado con Shopify (por ejemplo, uno local), puedes comprar la etiqueta directamente con ellos y luego añadir el número de seguimiento manualmente al preparar el pedido en Shopify.
* El seguimiento se sincroniza automáticamente cuando compras la etiqueta dentro de Shopify, y el cliente recibe la notificación.
